# lavaGameEngine
A simple "2.5D" game engine with a small fighting game built on top.

**Requries** the external libraries:
- GLFW
- Asimp
- SoLoud.

## Controls:
###### Player 1 (keyboard):
Move: WASD
Jump: Spacebar
Shoot: F
Throw: Q

###### Player 2 (Gamepad):
Move: DPad
Jump: A
Shoot: X
Throw: B
